# Github 实用技巧

## 一、搜索

### 1.1.搜索框聚焦

按 S 键，聚焦搜索框

### 1.2.搜索高级语法

比如 `springboot vue stars:>1000 push:>2024-01-02 language:java`

## 二、文件查看

### 2.1.仓库文件搜索

按 T 健，搜索仓库中的文件

### 2.2.行号跳转

按 L 健，跳转到指定行号。

### 2.3.行复制

点击行号，即可复制本行的内容

### 2.4.文件改动记录

按 B 键，查看文件改动记录。

## 三、代码阅读

按下“。”键，打开网页代码阅读器。

## 四、项目在线运行

在项目地址前，加上 `gitpod.io/#/` 前缀。

gitpod 会自动识别项目类型，并自动安装以来包。

可以输入命令来执行项目，并直接在网页中查看项目的运行效果。
